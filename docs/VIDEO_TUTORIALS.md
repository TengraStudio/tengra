# Video Tutorial Plan

This plan defines the first tutorial set and production checklist.

## Getting Started Guide

1. Install prerequisites and run `npm install`
2. Start dev environment with `npm run dev`
3. Open a project and send the first chat request
4. Explain where logs and settings are stored

Target length: 8-10 minutes

## Feature Walkthroughs

1. Chat + model switching
2. Project workspace and file tooling
3. Terminal and SSH integration
4. Changelog and localization workflow

Target length per video: 6-12 minutes

## Advanced Usage Scenarios

1. Multi-model orchestration and fallback strategy
2. Prompt templates and reusable workflows
3. IPC debugging and troubleshooting
4. Security hardening checks before release

Target length per video: 10-15 minutes

## Production Checklist

1. Script file in `docs/video-scripts/<topic>.md`
2. Demo repository and deterministic seed data
3. Recorded voice-over + captions
4. Linked resources (docs, commands, source paths)
5. Version tag in title (e.g., `v1.0.x`)

## Publishing Cadence

- Weekly: 1 getting-started or feature video
- Monthly: 1 advanced scenario
- Re-record when release introduces breaking UX/API changes
