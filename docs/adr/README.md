# Architecture Decision Records (ADR)

This folder tracks high-impact technical decisions in a durable format.

## Status

- `accepted`: Active decision for current architecture.
- `superseded`: Replaced by a newer ADR.
- `proposed`: Under review, not yet committed.

## Naming

Use `NNNN-title.md` where `NNNN` is a zero-padded sequence.

## Template

Use `0000-template.md` when creating a new ADR.
