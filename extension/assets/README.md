# Browser Extension Icons

This folder should contain the following icon files:

- `icon16.png` (16x16 pixels)
- `icon32.png` (32x32 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Creating Icons

You can create these icons from the Tandem logo:

1. Use the logo from `resources/icon.ico` or `logo_white.png`
2. Resize to each required size
3. Save as PNG files in this directory

## Online Tools

- [Favicon Generator](https://favicon.io/)
- [Icon Generator](https://realfavicongenerator.net/)
- [PNG Resizer](https://www.iloveimg.com/resize-image/resize-png)

## Temporary Solution

For development, you can use simple colored squares as placeholders until proper icons are created.
